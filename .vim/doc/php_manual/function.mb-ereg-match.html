mb_ereg_match

(PHP 4 >= 4.2.0, PHP 5)

mb_ereg_match ― マルチバイト文字列が正規表現に一致するか調べる

説明

bool mb_ereg_match ( string $pattern , string $string [, string $option ] )

mb_ereg_match()は、 string が正規表現 pattern に一致する場合にTRUE 、 そうでない場合に FALSE を返します。

文字エンコーディングとして、内部エンコーディングまたは mb_regex_encoding()で指定した 文字エンコーディングが使用されます。

mb_regex_encoding(), mb_ereg() も参照ください。
