hw_GetChildDocColl

(PHP 4)

hw_GetChildDocColl ― コレクションの子ドキュメントのオブジェクト ID

説明

array hw_getchilddoccoll ( int $connection , int $objectID )

コレクションの子ドキュメントに関するオブジェクト ID の配列を 返します。

hw_children() および hw_getchildcoll() も参照ください。
