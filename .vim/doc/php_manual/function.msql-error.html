msql_error

(PHP 4, PHP 5)

msql_error ― 最後の msql コールに関するエラーメッセージを返す

説明

string msql_error ( void )

msql_error() は、mSQL サーバで最後に発生したエラーを 返します。msql_error() でアクセス可能なのは 直近のエラーメッセージだけであることに注意しましょう。

返り値

直近のエラーメッセージを返します。エラーが発生しなかった場合は 空文字列を返します。
