is_binary

(No version information available, might be only in CVS)

is_binary ― 変数がネイティブバイナリ文字列かどうかを調べる

説明

bool is_binary ( mixed $var )

指定した変数が、ネイティブのバイナリ文字列かどうかを調べます。

パラメータ

var

調べる変数。

返り値

var がネイティブバイナリ文字列である場合に TRUE、それ以外の場合に FALSE を返します。
