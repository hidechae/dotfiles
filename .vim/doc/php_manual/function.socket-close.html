socket_close

(PHP 4 >= 4.0.7, PHP 5)

socket_close ― ソケットリソースを閉じる

説明

void socket_close ( resource $socket )

socket_close() は、 socket で指定したソケットリソースを閉じます。 この関数はソケット専用であり、その他のリソースに対しては用いることができません。

パラメータ

socket

socket_create() あるいは socket_accept() で作成したソケットリソース。

返り値

値を返しません。

  • socket_create()
  • socket_strerror()
