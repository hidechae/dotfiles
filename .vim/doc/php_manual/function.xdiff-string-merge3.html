xdiff_string_merge3

(PECL xdiff:0.2-1.4)

xdiff_string_merge3 ― 3 つの文字列をひとつに統合する

説明

mixed xdiff_string_merge3 ( string $str1 , string $str2 , string $str3 [, string &$error ] )

三つの文字列をひとつに結合します。

パラメータ

str1

str2

str3

error

指定した場合、パッチを拒否された部分がこの変数に保存されます。

返り値

統合された文字列を返します。内部エラーが発生した場合は FALSE、 統合された文字列が空の文字列の場合は TRUE を返します。
