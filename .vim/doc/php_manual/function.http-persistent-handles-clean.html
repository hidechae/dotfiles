http_persistent_handles_clean

(PECL pecl_http:1.5.0-1.5.5)

http_persistent_handles_clean ― 持続ハンドルを消去する

説明

string http_persistent_handles_clean ([ string $ident ] )

持続ハンドルを消去します (閉じます)。オプションで、ident による認証を行います。

パラメータ

clean

認証文字列。

返り値

値を返しません。
