ovrimos_field_len

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_field_len ― 出力カラムの長さを返す

説明

int ovrimos_field_len ( int $result_id , int $field_number )

指定した出力カラムの長さを取得します。

パラメータ

result_id

ovrimos_execute() あるいは ovrimos_exec() が返す結果 ID。

field_number

フィールド番号。1 から始まります。

返り値

フィールドの長さを表す整数値、あるいはエラー時に FALSE を返します。
