restore_exception_handler

(PHP 5)

restore_exception_handler ― 以前の例外ハンドラ関数を回復する

説明

bool restore_exception_handler ( void )

set_exception_handler() を使用して例外ハンドラ関数を 変更した後、元の例外ハンドラ(組込またはユーザ定義関数)に戻すために 使用されます。

返り値

この関数は常に TRUE を返します。
