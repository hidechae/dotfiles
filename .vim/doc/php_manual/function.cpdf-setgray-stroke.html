cpdf_setgray_stroke

(PHP 4, PHP 5 <= 5.0.5)

cpdf_setgray_stroke ― 描画色をグレー値に設定する

説明

bool cpdf_setgray_stroke ( int $pdf_document , float $gray_value )

現在の描画色を、指定したグレー値に設定します。

パラメータ

pdf_document

cpdf_open() が返す、ドキュメントのハンドル。

gray_value

グレー値を表す float 値。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
