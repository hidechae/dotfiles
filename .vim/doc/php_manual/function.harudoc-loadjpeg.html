HaruDoc::loadJPEG

(PECL haru:0.0.1)

HaruDoc::loadJPEG ― JPEG 画像を読み込み、新しい HaruImage のインスタンスを返す

説明

object HaruDoc::loadJPEG ( string $filename )

指定した JPEG 画像を読み込み、新しい HaruImage のインスタンスを返します。

パラメータ

filename

有効な JPEG 画像ファイル。

エラー / 例外

エラー時に HaruException をスローします。
