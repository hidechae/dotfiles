debugger_on

(No version information available, might be only in CVS)

debugger_on ― PHP の内部デバッガを有効にする (PHP 3)

説明

int debugger_on ( string $address )

PHP の内部デバッガを有効にし、 address に接続します。

パラメータ

address

デバッガのホスト名。

返り値

成功した場合に 1、エラー時に 0 を返します。

注意

この関数は PHP 3 でのみ有効です。
