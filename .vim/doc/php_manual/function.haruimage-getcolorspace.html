HaruImage::getColorSpace

(PECL haru:0.0.1)

HaruImage::getColorSpace ― 色空間の名前を取得する

説明

string HaruImage::getColorSpace ( void )

画像の色空間の名前を返します。名前は以下のいずれかとなります。

  • "DeviceGray"
  • "DeviceRGB"
  • "DeviceCMYK"
  • "Indexed"

エラー / 例外

エラー時に HaruException をスローします。
