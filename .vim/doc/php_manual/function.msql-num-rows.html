msql_num_rows

(PHP 4, PHP 5)

msql_num_rows ― 結果における行の数を得る

説明

int msql_num_rows ( resource $query_identifier )

msql_num_rows() は、結果セットにおける 行の数を返します。

パラメータ

result

評価された結果 リソース。 この結果は、msql_query() のコールにより得られたものです。

返り値

結果セットにおける行の数を返します。
