fbsql_warnings

(PHP 4 >= 4.0.6, PHP 5)

fbsql_warnings ― FrontBase 警告を有効または無効にする

説明

bool fbsql_warnings ([ bool $OnOff ] )

FrontBase の警告を有効または無効にします。

パラメータ

OnOff

警告を有効にするかしないか。

返り値

警告をオンにした場合は TRUE、そうでない場合は FALSE を返します。
