HaruDoc::getPageMode

(PECL haru:0.0.1)

HaruDoc::getPageMode ― 現在のページモードを取得する

説明

int HaruDoc::getPageMode ( void )

現在ドキュメントに設定されているページモードを返します。 とりうる値の一覧については HaruDoc::setPageMode() を参照ください。

エラー / 例外

エラー時に HaruException をスローします。
