maxdb_param_count

(PECL maxdb:1.0)

maxdb_param_count ― maxdb_stmt_param_count() のエイリアス

説明

この関数は次の関数のエイリアスです。 maxdb_stmt_param_count().

この関数エイリアスは非推奨であり、下位互換性維持のために残されています。 今後、PHP から削除される可能性がありますので、この関数を使用しないことを推奨します。
