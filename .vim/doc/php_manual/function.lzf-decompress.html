lzf_decompress

(PECL lzf:0.1-1.4)

lzf_decompress ― LZF 圧縮を解凍する

説明

string lzf_decompress ( string $data )

lzf_decompress() は、指定された data を解凍します。

パラメータ

data

圧縮されたデータ。

返り値

解凍されたデータを返します。エラー時には FALSE を返します。
