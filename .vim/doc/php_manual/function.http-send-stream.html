http_send_stream

(PECL pecl_http:0.1.0-1.5.5)

http_send_stream ― ストリームを送信する

説明

bool http_send_stream ( resource $stream )

(複数の) range リクエストをサポートする、 オープン済みのストリームを送信します。

パラメータ

stream

読み込み元となるストリーム (シーク可能でなければなりません)。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
