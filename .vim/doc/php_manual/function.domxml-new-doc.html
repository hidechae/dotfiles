domxml_new_doc

(PHP 4 >= 4.2.0)

domxml_new_doc ― 空の新規 XMLドキュメントを作成する

説明

DomDocument domxml_new_doc ( string $version )

スクラッチから新規 DOM ドキュメントを作成し、返します。

パラメータ

version

文章の XML バージョン番号

返り値

新規 DomDocument インスタンスを返します。
