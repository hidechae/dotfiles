HttpDeflateStream::update

(PECL pecl_http:0.21.0-1.5.5)

HttpDeflateStream::update ― 圧縮ストリームを更新する

説明

public string HttpDeflateStream::update ( string $data )

圧縮ストリームに、さらにデータを渡します。

パラメータ

data

圧縮するデータ。

返り値

成功した場合は圧縮したデータ、失敗した場合は FALSE を返します。
