ibase_blob_cancel

(PHP 4, PHP 5)

ibase_blob_cancel ― blob の生成を取り消す

説明

bool ibase_blob_cancel ( resource $blob_handle )

この関数は、まだ ibase_blob_close() で閉じられていない場合に BLOB を捨てます。

パラメータ

blob_handle

ibase_blob_create() でオープンした blob ハンドル。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

  • ibase_blob_import()
