tidyNode->isHtml

(PHP 5 >= 5.0.1)

tidyNode->isHtml ― このノードが HTML ドキュメントの場合 true を返す

説明

bool tidyNode->isHtml ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

注意: この関数は、PHP 4/Tidy 1 では tidy_node->is_html() という名前でした。
