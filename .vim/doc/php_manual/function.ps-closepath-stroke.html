ps_closepath_stroke

(PECL ps:1.1.0-1.3.5)

ps_closepath_stroke ― パスを閉じ、描画する

説明

bool ps_closepath_stroke ( resource $psdoc )

パスの終了点を開始点とつなげ、出来上がった線を描画します。

パラメータ

psdoc

ps_new() が返す、postscript ファイルのリソース ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
