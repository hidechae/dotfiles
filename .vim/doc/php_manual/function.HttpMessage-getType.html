HttpMessage::getType

(PECL pecl_http:0.10.0-1.5.5)

HttpMessage::getType ― メッセージの型を取得する

説明

public int HttpMessage::getType ( void )

メッセージの型を取得します。 HTTP_MSG_NONE、HTTP_MSG_REQUEST あるいは HTTP_MSG_RESPONSE のいずれかです。

返り値

HttpMessage::TYPE を返します。
