pfpro_cleanup

(PHP 4 >= 4.0.2, PHP 5 <= 5.0.5)

pfpro_cleanup ― Payflow Pro ライブラリをシャットダウンする

説明

bool pfpro_cleanup ( void )

Used to shutdown the Payflow Pro library cleanly.

この関数は、トランザクションを 終了した後スクリプトが終了する前にコールしなければなりません。 しかしこの関数のコールは省略することができ、その場合 この拡張モジュールはスクリプト終了時に pfpro_cleanup() を自動的にコールします。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
