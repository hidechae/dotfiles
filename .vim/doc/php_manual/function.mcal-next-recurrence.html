mcal_next_recurrence

(PHP 4)

mcal_next_recurrence ― イベントが次に発生する日を返す

説明

int mcal_next_recurrence ( int $stream , int $weekstart , array $next )

mcal_next_recurrence() は、指定した日以降に 次のイベントが発生する日を有するオブジェクトを返します。 イベントが発生しないか、何か問題がある場合は空の date フィールドを 返します。weekstart により週の初めとなる曜日を指定することができます。
