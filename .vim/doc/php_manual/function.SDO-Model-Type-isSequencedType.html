警告

この関数は、 実験的 なものです。この関数の動作・ 名前・その他ドキュメントに書かれている事項は、予告なく、将来的な PHP のリリースにおいて変更される可能性があります。 この関数は自己責任で使用してください。

この型がシーケンス型であるかどうかを調べる この型がシーケンス型である場合に TRUE、それ以外の場合に FALSE を返します。 シーケンス型は複数のプロパティをまたがって要素の順序を管理することが 可能で、非構造化テキストを含むことができます。シーケンス型についての 詳細な情報は、 シーケンスデータオブジェクトの 使用 を参照ください。

なし。

返り値

この型がシーケンス型である場合に TRUE、それ以外の場合に FALSE を返します。
