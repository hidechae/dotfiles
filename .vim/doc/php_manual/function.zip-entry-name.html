zip_entry_name

(PHP 4 >= 4.0.7, PHP 5 >= 5.2.0, PECL zip:1.0-1.9.0)

zip_entry_name ― ディレクトリエントリの名前を取得する

説明

string zip_entry_name ( resource $zip_entry )

指定したディレクトリエントリの名前を返します。

パラメータ

zip_entry

zip_read()により返されたディレクトリエントリ

返り値

ディレクトリエントリの名前

参考

  • zip_open()
  • zip_read()
