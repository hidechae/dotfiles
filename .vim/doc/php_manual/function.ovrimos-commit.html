ovrimos_commit

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_commit ― トランザクションをコミットする

説明

bool ovrimos_commit ( int $connection_id )

トランザクションをコミットします。

パラメータ

connection_id

ovrimos_connect() が返す Ovrimos 接続 ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

参考

  • ovrimos_rollback()
