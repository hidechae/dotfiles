enchant_dict_check

(PECL enchant:0.1-1.0.1)

enchant_dict_check ― 単語のスペルが正しいかどうかを調べる

説明

bool enchant_dict_check ( resource $dict , string $word )

単語のスペルが正しい場合に TRUE、 そうでない場合に FALSE を返します。

パラメータ

dict

辞書リソース。

word

調べる単語。

返り値

単語のスペルが正しい場合に TRUE、 そうでない場合に FALSE を返します。
