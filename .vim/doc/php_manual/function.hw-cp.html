hw_cp

(PHP 4)

hw_cp ― オブジェクトをコピーする

説明

int hw_cp ( int $connection , array $object_id_array , int $destination_id )

2番名のパラメータとして指定されたオブジェクトID を有するオブジェ クト ID destination id を有するコレクション にコピーします。

返される値は、コピーされたオブジェクトの数です。

hw_mv() も参照ください。
