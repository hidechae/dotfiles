openal_context_process

(PECL openal:0.1)

openal_context_process ― 指定したコンテキストを処理する

説明

bool openal_context_process ( resource $context )

パラメータ

context

Open AL (コンテキスト) リソース (事前に openal_context_create() で作成したもの)。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
