HaruDoc::output

(PECL haru:0.0.1)

HaruDoc::output ― ドキュメントデータを出力バッファに書き出す

説明

bool HaruDoc::output ( void )

ドキュメントデータを標準出力に書き出します。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
