sin

(PHP 4, PHP 5)

sin ― 正弦（サイン）

説明

float sin ( float $arg )

sin() は、arg のサインを 返します。arg はラジアンです。

パラメータ

arg

ラジアンで表した値。

返り値

arg のサインを返します。

例

Example#1 sin() の例

<?php

// 結果の精度は、指定によります。
echo sin(deg2rad(60));  //  0.866025403 ...
echo sin(60);           // -0.304810621 ...

?>
