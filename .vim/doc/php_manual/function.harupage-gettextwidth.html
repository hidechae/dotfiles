HaruPage::getTextWidth

(PECL haru:0.0.1)

HaruPage::getTextWidth ― 現在のフォントサイズ、文字エンコーディングおよび単語間隔を使用してテキストの幅を取得する

説明

float HaruPage::getTextWidth ( string $text )

現在のフォントサイズ、文字エンコーディングおよび単語間隔を使用して テキストの幅を返します。

パラメータ

text

調べたいテキスト。

エラー / 例外

エラー時に HaruException をスローします。
