dblist

(PHP 4)

dblist ― 使用されている DBM 互換ライブラリの一覧を返す

説明

string dblist ( void )

GDBM のバージョンを返します。

返り値

GDBM のバージョンを表す文字列を返します。

例

Example#1 コマンドラインで情報を取得する

[marcus@zaphod marcus]$ php -r 'echo dblist();'
This is GDBM version 1.8.0, as of May 19, 1999.
