mb_strimwidth

(PHP 4 >= 4.0.6, PHP 5)

mb_strimwidth ― 指定した幅で文字列を丸める

説明

string mb_strimwidth ( string $str , int $start , int $width [, string $trimmarker [, string $encoding ]] )

mb_strimwidth() は、文字列 str を指定した幅 width で丸めます。 この関数は、丸められた文字列を返します。

trimmarker が設定された場合、 trimmarker が丸められた文字列に追加されます。

start は開始位置のオフセットであり、 文字列の始めからの文字数です。(最初の文字は 0)

trimmarker は文字列であり、 文字列が丸められた場合にその文字列の最後に追加されます。

encoding は文字エンコーディングであり、 省略された場合は内部エンコーディングが使用されます。

Example#1 mb_strimwidth() の例

<?php
$str = mb_strimwidth($str, 0, 40, "..>");
?>

mb_strwidth()、 mb_internal_encoding() も参照ください。
