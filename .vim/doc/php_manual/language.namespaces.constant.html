__NAMESPACE__

コンパイル時に、現在の名前空間を表す定数 __NAMESPACE__ が定義されます。 名前空間の外部では、この定数の値は空文字列となります。 この定数は、名前空間内のローカル名から完全な名前を作成する際に便利です。

Example#1 __NAMESPACE__ の使用法

<?php
namespace A::B::C;
         
function foo() {
// 何かの処理
}

set_error_handler(__NAMESPACE__ . "::foo");
?>
