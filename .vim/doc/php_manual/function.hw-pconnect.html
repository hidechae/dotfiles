hw_pConnect

(PHP 4)

hw_pConnect ― 持続的データベース接続を作成する

説明

int hw_pconnect ( string $host , int $port [, string $username ], string $password )

成功時に接続インデックスを、接続できなかった場合に FALSE を返します。 Hyperwave サーバへの持続的接続をオープンします。 各引数は、ポート番号を除き引用符で括った文字列である必要があります。 username とpassword 引数は オプションであり、省略することができます。この場合、サーバとの認証は 行われません。これは、匿名ユーザとして認証されることに似ています。 この関数は、他の Hyperwave 関数で必要な接続インデックスを返します。 同時に複数の持続的接続をオープンすることができます。

hw_connect() も参照ください。
