HaruPage::setFlatness

(PECL haru:0.0.1)

HaruPage::setFlatness ― ページの平坦度を設定する

説明

bool HaruPage::setFlatness ( float $flatness )

ページの平坦度を定義します。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
