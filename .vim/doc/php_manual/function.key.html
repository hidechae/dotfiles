key

(PHP 4, PHP 5)

key ― 連想配列からキーを取り出す

説明

mixed key ( array &$array )

key() は、 現在の配列位置における連想配列要素のキーを返します。

Example#1 key() の例

<?php
$array = array(
    'fruit1' => 'apple',
    'fruit2' => 'orange',
    'fruit3' => 'grape',
    'fruit4' => 'apple',
    'fruit5' => 'apple');

// このループは値が "apple" である
// 全ての連想配列のキーを表示する
while ($fruit_name = current($array)) {
    if ($fruit_name == 'apple') {
        echo key($array).'<br />';
    }
    next($array);
}
?>

current() および next() も参照ください。
