tidy_get_release

(PHP 5, PECL tidy:0.5.2-1.2)

tidy_get_release ― Tidy ライブラリのリリース日 (バージョン) を取得する

説明

手続き言語型スタイル:

string tidy_get_release ( void )

オブジェクト指向言語型スタイル:

string tidy->getRelease ( void )

この関数は、Tidy ライブラリのリリース日を文字列で返します。
