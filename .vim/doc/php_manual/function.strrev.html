strrev

(PHP 4, PHP 5)

strrev ― 文字列を逆順にする

説明

string strrev ( string $string )

string を逆順にして返します。

パラメータ

string

逆順にしたい文字列。

返り値

逆順にした文字列を返します。

例

Example#1 strrev() で文字列を逆順にする

<?php
echo strrev("Hello world!"); // 出力は "!dlrow olleH" となります
?>
