end

(PHP 4, PHP 5)

end ― 配列の内部ポインタを最終要素にセットする

説明

mixed end ( array &$array )

end() は array の内部ポインタを最後の要素まで進め、その値を返します。

Example#1 end() の簡単な例

<?php

$fruits = array('apple', 'banana', 'cranberry');
echo end($fruits); // cranberry

?>

current()、 each()、 prev()、 next() および reset() も参照ください。
