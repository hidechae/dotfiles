fdf_close

(PHP 4, PHP 5)

fdf_close ― FDF ドキュメントを閉じる

説明

void fdf_close ( resource $fdf_document )

FDF ドキュメントを閉じます。

パラメータ

fdf_document

fdf_create()、 fdf_open() あるいは fdf_open_string() が返す FDF ドキュメントハンドル。

返り値

値を返しません。
