cpdf_set_title

(PHP 4, PHP 5 <= 5.0.5)

cpdf_set_title ― pdf ドキュメントの title フィールドを設定する

説明

bool cpdf_set_title ( int $pdf_document , string $title )

PDF ドキュメントの title を設定します。

パラメータ

pdf_document

cpdf_open() が返す、ドキュメントのハンドル。

title

ドキュメントのタイトル。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
