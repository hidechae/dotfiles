dngettext

(PHP 4 >= 4.2.0, PHP 5)

dngettext ― dgettext の複数形版

説明

string dngettext ( string $domain , string $msgid1 , string $msgid2 , int $n )

dngettext() 関数により、単一の複数形メッセージ参照について 現在のドメインを上書きすることができます。

パラメータ

domain

ドメイン。

msgid1

msgid2

n

返り値

成功した場合に文字列を返します。
