imap_8bit

(PHP 4, PHP 5)

imap_8bit ― 8 ビット文字列を quoted-printable 文字列に変換する

説明

string imap_8bit ( string $string )

(» RFC2045, section 6.7 に基づき) 8 ビット文字列を quoted-printable 文字列に変換します。

パラメータ

string

変換する 8 ビット文字列。

返り値

quoted-printable 文字列を返します。
