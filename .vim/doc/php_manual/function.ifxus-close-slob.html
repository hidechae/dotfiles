ifxus_close_slob

(No version information available, might be only in CVS)

ifxus_close_slob ― SLOB オブジェクトを削除する

説明

bool ifxus_close_slob ( int $bid )

指定した SLOB オブジェクト ID bid の SLOB オブジェクトを削除します。

パラメータ

bid

既存の SLOB の ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
