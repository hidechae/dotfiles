DOMElement->removeAttribute() ― 属性を削除する

説明

DOMElement
bool removeAttribute ( string $name )

name という名前の属性を要素から削除します。

パラメータ

name

属性の名前。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

エラー / 例外

DOM_NO_MODIFICATION_ALLOWED_ERR

ノードが読み込み専用の場合に発生します。
