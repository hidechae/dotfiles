tan

(PHP 4, PHP 5)

tan ― 正接（タンジェント）

説明

float tan ( float $arg )

tan() は、arg のタンジェントを 返します。arg はラジアンです。

パラメータ

arg

処理する引数をラジアンで表したもの。

返り値

arg のタンジェントを返します。

例

Example#1 tan() の例

<?php

echo tan(M_PI_4); // 1

?>
