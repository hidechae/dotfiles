odbc_field_name

(PHP 4, PHP 5)

odbc_field_name ― カラム名を得る

説明

string odbc_field_name ( resource $result_id , int $field_number )

odbc_field_name() は、指定された ODBC 結果 ID において指定されたカラム番号にあるフィールドの名前を返します。 フィールド番号は、1 から始まります。エラーの場合、 FALSE を返します。
