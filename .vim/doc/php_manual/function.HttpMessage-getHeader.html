HttpMessage::getHeader

(PECL pecl_http:1.1.0-1.5.5)

HttpMessage::getHeader ― ヘッダを取得する

説明

public string HttpMessage::getHeader ( string $header )

メッセージのヘッダを取得します。

パラメータ

header

ヘッダの名前。

返り値

成功した場合はヘッダの値、ヘッダが存在しない場合は NULL を返します。
