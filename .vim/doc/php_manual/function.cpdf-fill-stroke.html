cpdf_fill_stroke

(PHP 4, PHP 5 <= 5.0.5)

cpdf_fill_stroke ― 現在のパスを塗りつぶし、描く

説明

bool cpdf_fill_stroke ( int $pdf_document )

現在のパスの内部を現在の塗りつぶし色で塗りつぶし、 それを描画します。

パラメータ

pdf_document

cpdf_open() が返す、ドキュメントのハンドル。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
