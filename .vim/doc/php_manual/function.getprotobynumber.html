getprotobynumber

(PHP 4, PHP 5)

getprotobynumber ― プロトコル番号が指すプロトコル名を取得する

説明

string getprotobynumber ( int $number )

getprotobynumber() は、 /etc/protocols に基づき プロトコル番号 number が指すプロトコル名を返します。

パラメータ

number

プロトコル番号。

返り値

プロトコル名を文字列で返します。
