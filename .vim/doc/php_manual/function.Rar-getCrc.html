Rar::getCrc ― エントリの CRC を取得する

説明

Rar
int getCrc ( void )

Rar::getCrc() は、アーカイブエントリの CRC を返します。

返り値

アーカイブエントリの CRC、あるいはエラー時に FALSE を返します。
