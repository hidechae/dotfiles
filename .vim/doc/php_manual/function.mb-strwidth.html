mb_strwidth

(PHP 4 >= 4.0.6, PHP 5)

mb_strwidth ― 文字列の幅を返す

説明

int mb_strwidth ( string $str [, string $encoding ] )

mb_strwidth() は文字列 str の幅を返します。

マルチバイト文字は通常シングルバイト文字の倍の幅となります。

文字の幅
文字 幅
U+0000 - U+0019 0
U+0020 - U+1FFF 1
U+2000 - U+FF60 2
U+FF61 - U+FF9F 1
U+FFA0 - 2

encoding は文字エンコーディングであり、 省略された場合は内部エンコーディングが使用されます。

mb_strimwidth(), mb_internal_encoding() も参照ください。
