newt_finished

(PECL newt:0.1-1.1)

newt_finished ― newt インターフェースを終了する

説明

int newt_finished ( void )

newt インターフェースを終了します。プログラムが終了する際にこの関数を コールします。

返り値

成功した場合に 1、失敗した場合に 0 を返します。
