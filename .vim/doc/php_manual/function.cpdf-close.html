cpdf_close

(PHP 4, PHP 5 <= 5.0.5)

cpdf_close ― pdf ドキュメントを閉じる

説明

bool cpdf_close ( int $pdf_document )

PDF ドキュメントを閉じます。

この関数は、 cpdf_finalize(), cpdf_output_buffer(), cpdf_save_to_file() の後で、最後にコールする必要があります。

パラメータ

pdf_document

cpdf_open() が返す、ドキュメントのハンドル。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
