filepro_fieldwidth

(PHP 4, PHP 5 <= 5.1.6)

filepro_fieldwidth ― フィールド幅を取得する

説明

int filepro_fieldwidth ( int $field_number )

field_number に対応するフィールドの幅を返します。

パラメータ

field_number

フィールド番号。

返り値

フィールドの幅を表す整数値、あるいはエラーの場合に FALSE を返します。
