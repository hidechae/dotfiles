OCI-Lob->size

(PHP 5, PECL oci8:1.1-1.2.4)

OCI-Lob->size ― ラージオブジェクトのサイズを返す

説明

OCI-Lob
int size ( void )

ラージオブジェクトのサイズを取得します。

返り値

ラージオブジェクトのサイズを返します。 エラーの場合、FALSE を返します。 空のオブジェクトは長さ 0 となります。
