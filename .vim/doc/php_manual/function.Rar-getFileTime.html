(No version information available, might be only in CVS)

Rar::getFileTime ― エントリの最終更新時刻を取得する

説明

Rar
string getFileTime ( void )

エントリの最終更新時刻を取得します。

返り値

エントリの最終更新時刻を YYYY-MM-DD HH:II:SS 形式で表した文字列、あるいはエラー時に FALSE を返します。
