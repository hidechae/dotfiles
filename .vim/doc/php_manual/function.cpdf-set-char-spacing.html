cpdf_set_char_spacing

(PHP 4, PHP 5 <= 5.0.5)

cpdf_set_char_spacing ― 文字間隔を設定する

説明

bool cpdf_set_char_spacing ( int $pdf_document , float $space )

文字の間隔を設定します。

パラメータ

pdf_document

cpdf_open() が返す、ドキュメントのハンドル。

space

文字列内の個々の文字の間隔。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
