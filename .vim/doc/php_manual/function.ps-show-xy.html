ps_show_xy

(PECL ps:1.1.0-1.3.5)

ps_show_xy ― 指定された位置にテキストを出力する

説明

bool ps_show_xy ( resource $psdoc , string $text , float $x , float $y )

指定された位置にテキストを出力します。

パラメータ

psdoc

ps_new() が返す、postscript ファイルのリソース ID。

text

出力するテキスト。

x

テキストを囲むボックスの左下角の x 座標。

y

テキストを囲むボックスの左下角の y 座標。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
