sqlite_error_string

(PHP 5, PECL sqlite:1.0-1.0.3)

sqlite_error_string ― エラーコードの説明を返す

説明

string sqlite_error_string ( int $error_code )

sqlite_last_error()から返される error_code の可読性が高い説明を返します。
