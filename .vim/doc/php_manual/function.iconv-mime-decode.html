iconv_mime_decode

(PHP 5)

iconv_mime_decode 〗 MIME ヘッダフィ〖ルドをデコ〖ドする

棱汤

string iconv_mime_decode ( string $encoded_header [, int $mode [, string $charset ]] )

MIME ヘッダフィ〖ルドをデコ〖ドします。

パラメ〖タ

encoded_header

エンコ〖ドされたヘッダを山す矢机误。

mode

mode は、iconv_mime_decode() が稍赖な妨及の MIME ヘッダフィ〖ルドに柳而した眷圭の 慷る神いを年盗します。笆布のビットマスクの寥み圭わせで回年が材墙です。

iconv_mime_decode() で回年できるビットマスク
猛 年眶叹 棱汤
1 ICONV_MIME_DECODE_STRICT 回年すると、ヘッダは » RFC2047 で年盗されている筛洁に窗链洁凋する妨及でデコ〖ドされます。 このオプションはデフォルトでは痰跟になっています。なぜなら、坤の面には おかしなメ〖ルソフトが驴く赂哼し、それらは惮呈に骄わずに粗般った MIME ヘッダを栏喇するからです。
2 ICONV_MIME_DECODE_CONTINUE_ON_ERROR 回年すると、iconv_mime_decode_headers() は矢恕弄なエラ〖を痰浑し、デコ〖ド侯度を费鲁します。

charset

オプションの charset パラメ〖タは、冯蔡の 矢机セットを回年します。回年されなかった眷圭は iconv.internal_encoding が脱いられます。

手り猛

喇根した眷圭はデコ〖ドされた MIME フィ〖ルドを、 デコ〖ド面にエラ〖が券栏した眷圭は FALSE を手します。

毋

Example#1 iconv_mime_decode() の毋

<?php
// この冯蔡は "Subject: Prfung Prfung" となります。
echo iconv_mime_decode("Subject: Prfung Prfung",
                       0, "ISO-8859-1");
?>
