vsprintf

(PHP 4 >= 4.0.7, PHP 5)

vsprintf ― フォーマットされた文字列を返す

説明

string vsprintf ( string $format , array $args )

sprintf()と動作は同じですが、 可変長の引数ではなく配列を引数とします。

パラメータ

format

args

返り値

(sprintf()のドキュメントに記述された) format に基づきフォーマットされた文字列として配列値を返します。
