curl_error

(PHP 4 >= 4.0.3, PHP 5)

curl_error ― 現在のセッションに関する直近のエラー文字列を返す

説明

string curl_error ( resource $ch )

直近の cURL 操作に関するエラーメッセージをクリアテキストで返します。

パラメータ

ch

curl_init() が返す cURL ハンドル。

返り値

エラー番号、あるいはエラーが発生しなかった場合は '' (空文字) を返します。
