sys_get_temp_dir

(PHP 5 >= 5.2.1)

sys_get_temp_dir ― 一時ファイル用に使用されるディレクトリのパスを返す

説明

string sys_get_temp_dir ( void )

PHP が一時ファイルを保存するデフォルトのディレクトリのパスを返します。

返り値

一時ディレクトリのパスを返します。

参考

  • tmpfile()
  • tempnam()
