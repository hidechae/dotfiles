XMLReader::moveToFirstAttribute

(PHP 5 >= 5.1.2)

XMLReader::moveToFirstAttribute ― 最初の属性にカーソルを移動する

説明

XMLReader
bool moveToFirstAttribute ( void )

最初の属性にカーソルを移動します。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
