mcal_date_compare

(PHP 4)

mcal_date_compare ― 二つの日付を比較する

説明

int mcal_date_compare ( int $a_year , int $a_month , int $a_day , int $b_year , int $b_month , int $b_day )

mcal_date_compare() は、二つの日付を比較し、 a<b , a==b, a>b の場合に、それぞれ、<0, 0, >0 を返します。
