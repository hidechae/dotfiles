quotemeta

(PHP 4, PHP 5)

quotemeta ― メタ文字をクォートする

説明

string quotemeta ( string $str )

文字列 str について、 

. \ + * ? [ ^ ] ( $ )
の前にバックスラッシュ文字 (\) でクォートして返します。

パラメータ

str

入力文字列。

返り値

メタ文字をクォートした文字列を返します。

注意

注意: この関数はバイナリデータに対応しています。
