stats_rand_gen_beta ― 無作為な値を生成する

説明

float stats_rand_gen_beta ( float $a , float $b )

パラメータ A および B で表されるベータ分布から、無作為な値を返します。 ベータ密度は、< x < に対して x^(a-1) * (1-x)^(b-1) / B(a,b) となります。これは R. C. H. Cheng の手法です。
