strcspn

(PHP 4, PHP 5)

strcspn ― マスクにマッチしない最初のセグメントの長さを返す

説明

int strcspn ( string $str1 , string $str2 [, int $start [, int $length ]] )

str1 において str2 の文字がどれも含まれていない最初のセグメントの長さを返します。

パラメータ

str1

str2

start

調べ始める位置。

length

調べる文字列の長さ。

返り値

セグメントの長さを返します。

変更履歴

バージョン 説明
4.3.0 start および length が追加されました。

注意

注意: この関数はバイナリデータに対応しています。
