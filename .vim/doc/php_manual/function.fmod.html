fmod

(PHP 4 >= 4.2.0, PHP 5)

fmod ― 引数で除算をした際の剰余を返す

説明

float fmod ( float $x , float $y )

被除数（x ）を除数（y ） で割った余りを返します。余り（r）は、 整数 i を使用して x = i * y + r で定義されます。 y がゼロ以外の場合はr は x と同符号で、絶対値は y より小さくなります。

パラメータ

x

被除数。

y

除数。

返り値

x /y の剰余を返します。

例

Example#1 fmod() の使用法

<?php
$x = 5.7;
$y = 1.3;
$r = fmod($x, $y);
// $r は 0.5 です。なぜなら 4 * 1.3 + 0.5 = 5.7 だからです。
?>
