cpdf_setdash

(PHP 4, PHP 5 <= 5.0.5)

cpdf_setdash ― 破線のパターンを設定する

説明

bool cpdf_setdash ( int $pdf_document , float $white , float $black )

破線のパターンを指定したものに設定します。両方のパラメータを 0 にした場合は実線となります。

パラメータ

pdf_document

cpdf_open() が返す、ドキュメントのハンドル。

white

白い部分の長さを表す float 値。

black

黒い部分の長さを表す float 値。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
