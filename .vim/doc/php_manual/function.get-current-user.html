get_current_user

(PHP 4, PHP 5)

get_current_user ― 現在の PHP スクリプトの所有者の名前を取得する

説明

string get_current_user ( void )

現在の PHP スクリプトの所有者の名前を返します。

返り値

ユーザ名を表す文字列。
