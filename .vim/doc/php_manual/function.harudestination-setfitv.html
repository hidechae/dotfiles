HaruDestination::setFitV

(PECL haru:0.0.1)

HaruDestination::setFitV ― ページの表示を、ウィンドウの高さにあわせるよう設定する

説明

bool HaruDestination::setFitV ( float $left )

ページの表示を、ウィンドウの高さにあわせるようにします。

パラメータ

left

ページの左端の位置。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
