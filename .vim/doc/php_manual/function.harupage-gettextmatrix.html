HaruPage::getTextMatrix

(PECL haru:0.0.1)

HaruPage::getTextMatrix ― そのページの現在のテキスト変換行列を取得する

説明

array HaruPage::getTextMatrix ( void )

そのページの現在のテキスト変換行列を、 "a"、"b"、"c"、"d"、"x" および "y" の 6 つの要素からなる配列で返します。

エラー / 例外

エラー時に HaruException を返します。
