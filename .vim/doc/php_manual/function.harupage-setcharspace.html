HaruPage::setCharSpace

(PECL haru:0.0.1)

HaruPage::setCharSpace ― ページの文字間隔を設定する

説明

bool HaruPage::setCharSpace ( float $char_space )

ページの文字間隔を定義します。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
