mb_strlen

(PHP 4 >= 4.0.6, PHP 5)

mb_strlen ― 文字列の長さを得る

説明

int mb_strlen ( string $str [, string $encoding ] )

mb_strlen() は、文字エンコーディング encoding の文字列 str の文字数を返します。 マルチバイト文字の一文字は1個として数えられます。

encoding は、str 用の文字エンコーディングです。 encoding が省略された場合、 内部文字エンコーディングが使用されます。

mb_internal_encoding()、 strlen() も参照ください。
