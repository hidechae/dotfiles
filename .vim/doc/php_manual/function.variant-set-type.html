variant_set_type

(PHP 5)

variant_set_type ― variant を「その場で」別の型に変換する

説明

void variant_set_type ( variant $variant , int $type )

この関数は variant_cast() に似ていますが、 「その場で」変換されるという点が異なります。新しい variant は 作成されません。この関数に渡すパラメータの意味は variant_cast() の場合と同じです。

variant_cast() も参照ください。
