lcg_value

(PHP 4, PHP 5)

lcg_value ― 複合線形合同法

説明

float lcg_value ( void )

lcg_value() は、(0, 1)の範囲の疑似乱数を返します。 この関数は、周期が 2^31 - 85 および 2^31 - 249 の 2 つの CG を組み合わせます。 この関数の周期はこれら 2 つの素数の積と等価です。

返り値

(0, 1) の範囲の疑似乱数を、浮動小数点数で返します。

  • mt_rand()
