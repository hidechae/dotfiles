closelog

(PHP 4, PHP 5)

closelog ― システムログへの接続を閉じる

説明

bool closelog ( void )

closelog() はシステムログへの書きこみに使用されているデスクプリタを閉じます。 closelog() の使用はオプションです。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

参考

  • define_syslog_variables()
  • syslog()
  • openlog()
