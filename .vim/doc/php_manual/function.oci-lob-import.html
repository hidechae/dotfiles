OCI-Lob->import

(PHP 5, PECL oci8:1.1-1.2.4)

OCI-Lob->import ― ファイルデータを LOB にインポートする

説明

OCI-Lob
bool import ( string $filename )

filename からのデータをラージオブジェクトの現在の位置に書き込みます。

パラメータ

filename

ファイルへのパス。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
