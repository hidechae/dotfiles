swf_shapefillbitmapclip

(PHP 4)

swf_shapefillbitmapclip ― 現在の塗りつぶしモードをクリップ付きビットマップに設定する

説明

void swf_shapefillbitmapclip ( int $bitmapid )

塗りつぶしをクリップ付きのビットマップに設定します。 空いた部分は、ビットマップで埋められます。

パラメータ

bitmapid

ビットマップ ID。

返り値

値を返しません。
