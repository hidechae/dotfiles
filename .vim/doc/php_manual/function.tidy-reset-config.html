tidy_reset_config

(PECL tidy:0.7-1.2)

tidy_reset_config ― Tidy の設定をデフォルト値に戻す

説明

bool tidy_reset_config ( void )

この関数は、Tidy の設定をデフォルト値に戻します。

注意: この関数は、Tidy 1.0 でのみ利用可能です。 この関数は、Tidy 2.0 では古い関数として削除されているためです。
