DOMXPath->registerNamespace() ― DOMXPath オブジェクトの名前空間を登録する

説明

DOMXPath
bool registerNamespace ( string $prefix , string $namespaceURI )

DOMXPath オブジェクトに、 namespaceURI および prefix を登録します。

パラメータ

prefix

プレフィックス。

namespaceURI

名前空間の URI。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
