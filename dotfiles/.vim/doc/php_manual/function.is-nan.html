is_nan

(PHP 4 >= 4.2.0, PHP 5)

is_nan ― 値が数値でないかどうかを判定する

説明

bool is_nan ( float $val )

val が '非数値 (not a number)' であるかどうかを調べます。たとえば acos(1.01) の結果などがこれにあたります。

パラメータ

val

調べる値。

返り値

val が '非数値 (not a number)' の場合に TRUE、そうでない場合に FALSE を返します。
