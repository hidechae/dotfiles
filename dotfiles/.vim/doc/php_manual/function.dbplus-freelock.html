dbplus_freelock

(PHP 4 >= 4.0.7, PECL dbplus:0.9)

dbplus_freelock ― タプルの書き込みロックを解放する

説明

int dbplus_freelock ( resource $relation , string $tuple )

dbplus_getlock() より以前に得た指定 tuple の書き込みロックを解放します。

パラメータ

relation

dbplus_open() でオープンしたリレーション。

tuple

返り値
