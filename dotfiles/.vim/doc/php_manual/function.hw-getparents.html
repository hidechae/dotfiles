hw_GetParents

(PHP 4)

hw_GetParents ― 親のオブジェクト ID

説明

array hw_getparents ( int $connection , int $objectID )

オブジェクト ID の添字配列を返します。各オブジェクト ID は、 ID objectID を有するオブジェクトの 親に属しています。
