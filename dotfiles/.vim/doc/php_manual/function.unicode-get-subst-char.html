unicode_get_subst_char ― 文字列変換エラー時に使用する置換文字を取得する

説明

unicode unicode_get_subst_char ( void )

unicode_set_subst_char() で設定した、 文字列変換エラー時に使用する置換文字を返します。

返り値

unicode 型の置換文字を返します。
