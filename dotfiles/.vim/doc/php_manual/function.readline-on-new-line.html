readline_on_new_line

(PHP 5 >= 5.1.0)

readline_on_new_line ― カーソルが新しい行に移動したことを readline に通知する

説明

void readline_on_new_line ( void )

readline に、カーソルが新しい行に移動したことを通知します。

返り値

値を返しません。
