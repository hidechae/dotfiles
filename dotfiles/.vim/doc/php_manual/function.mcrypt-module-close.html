mcrypt_module_close

(PHP 4 >= 4.0.2, PHP 5)

mcrypt_module_close ― mcrypt モジュールを閉じる

説明

bool mcrypt_module_close ( resource $td )

この関数は、指定した暗号化ハンドルを閉じます。

使用例は mcrypt_module_open() を参照ください。
