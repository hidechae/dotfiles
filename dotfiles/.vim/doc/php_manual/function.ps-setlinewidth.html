ps_setlinewidth

(PECL ps:1.1.0-1.3.5)

ps_setlinewidth ― 線幅を設定する

説明

bool ps_setlinewidth ( resource $psdoc , float $width )

以降の描画処理での線幅を設定します。

パラメータ

psdoc

ps_new() が返す、postscript ファイルのリソース ID。

width

線の幅 (ポイント数)。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
