cpdf_closepath

(PHP 4, PHP 5 <= 5.0.5)

cpdf_closepath ― パスを閉じる

説明

bool cpdf_closepath ( int $pdf_document )

現在のパスを閉じます。

パラメータ

pdf_document

cpdf_open() が返す、ドキュメントのハンドル。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
