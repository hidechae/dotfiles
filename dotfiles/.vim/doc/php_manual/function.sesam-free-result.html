sesam_free_result

(No version information available, might be only in CVS)

sesam_free_result ― クエリに関するリソースを開放する

説明

bool sesam_free_result ( string $result_id )

クエリのリソースを開放します。

パラメータ

result_id

sesam_query() が返す結果 ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
