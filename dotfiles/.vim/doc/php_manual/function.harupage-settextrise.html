HaruPage::setTextRise

(PECL haru:0.0.1)

HaruPage::setTextRise ― 現在のテキストライズの値を設定する

説明

bool HaruPage::setTextRise ( float $rise )

現在のテキストライズの値を定義します。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
