ob_flush

(PHP 4 >= 4.2.0, PHP 5)

ob_flush ― 出力バッファをフラッシュ(送信)する

説明

void ob_flush ( void )

この関数は、(ある場合に)出力バッファの内容を送信します。 更にバッファの内容を処理したい場合には、ob_flush() がコールされた後にバッファの内容が破棄されるので、 ob_flush() の前に ob_get_contents() をコールする必要があります。

この関数は、ob_end_flush() のように出力バッファを破棄しません。

返り値

値を返しません。
