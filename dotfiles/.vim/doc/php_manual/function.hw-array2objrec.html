hw_Array2Objrec

(PHP 4)

hw_Array2Objrec ― オブジェクト配列からオブジェクトレコードに属性を変換する

説明

string hw_array2objrec ( array $object_array )

object_array をオブジェクトレコードに変換します。 'Title' のような異なった言語で多重定義された属性は正しく処理されます。

hw_objrec2array() も参照ください。
