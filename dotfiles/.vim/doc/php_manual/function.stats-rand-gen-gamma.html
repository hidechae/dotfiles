stats_rand_gen_gamma

(PECL stats:1.0.0-1.0.2)

stats_rand_gen_gamma ― ガンマ分布から無作為な値を生成する

説明

float stats_rand_gen_gamma ( float $a , float $r )

密度 (A**R)/Gamma(R) * X**(R-1) * Exp(-A*X) のガンマ分布から、 無作為な値を生成します。

パラメータ

a

ガンマ分布の位置母数 (a > 0)。

r

ガンマ分布の形状母数 (r > 0)。

返り値
