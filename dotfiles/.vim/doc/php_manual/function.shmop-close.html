shmop_close

(PHP 4 >= 4.0.4, PHP 5)

shmop_close ― 共有メモリブロックを閉じる

説明

void shmop_close ( int $shmid )

shmop_close() は共有メモリブロックを閉じるために 使用されます。

パラメータ

shmid

shmop_open() が作成した共有メモリブロックの識別子。

返り値

値を返しません。

例

Example#1 共有メモリブロックを閉じる

<?php
shmop_close($shm_id);
?>

この例は、$shm_id を ID とする共有メモリブロックを閉じます。
