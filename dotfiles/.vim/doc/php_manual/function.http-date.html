http_date

(PECL pecl_http:0.1.0-1.5.5)

http_date ― HTTP の RFC に準拠した日付を作成する

説明

string http_date ([ int $timestamp ] )

RFC 1123 に準拠した、妥当な形式の HTTP 日付を作成します。 Wed, 22 Dec 2004 11:34:47 GMT のような形式になります。

パラメータ

timestamp

Unix タイムスタンプ。省略した場合は現在時刻。

返り値

HTTP の日付を文字列で返します。
