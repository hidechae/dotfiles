getmyuid

(PHP 4, PHP 5)

getmyuid ― PHP スクリプト所有者のユーザ ID を取得する

説明

int getmyuid ( void )

現在のスクリプトのユーザ ID を取得します。

返り値

現在のスクリプトのユーザ ID を返し、エラーの場合は FALSE を返します。
