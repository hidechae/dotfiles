mcal_time_valid

(PHP 4)

mcal_time_valid ― 指定した時間、分、秒が有効な時間である場合に TRUE を返す

説明

bool mcal_time_valid ( int $hour , int $minutes , int $seconds )

mcal_time_valid() は、指定した時間、分、秒が 有効な時間である場合に TRUE を、そうでない場合に FALSE を 返します。
