str_repeat

(PHP 4, PHP 5)

str_repeat ― 文字列を反復する

説明

string str_repeat ( string $input , int $multiplier )

input を multiplier 回を繰り返した文字列を返します。

パラメータ

input

繰り返す文字列。

multiplier

input を繰り返す回数。

multiplier は 0 以上でなければなりません。 multiplier が 0 に設定された場合、この関数は空文字を返します。

返り値

繰り返した文字列を返します。

返り値

Example#1 str_repeat() の例

<?php
echo str_repeat("-=", 10);
?>
