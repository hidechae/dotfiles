oci_cancel

(PHP 5, PECL oci8:1.1-1.2.4)

oci_cancel ― カーソルからの読み込みをキャンセルする

説明

bool oci_cancel ( resource $statement )

カーソルを無効にして関連付けられた全てのリソースを開放し、 読み込みをキャンセルします。

パラメータ

statement

OCI ステートメント

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
