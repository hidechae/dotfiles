OCI-Lob->close

(PHP 5, PECL oci8:1.1-1.2.4)

OCI-Lob->close ― LOB ディスクリプタを閉じる

説明

OCI-Lob
bool close ( void )

OCI-Lob->close() は LOB あるいは FILE のディスクリプタを閉じます。この関数は OCI-Lob->writeTemporary を併用した場合のみ使用されるべきです。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
