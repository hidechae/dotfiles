HttpDeflateStream::finish

(PECL pecl_http:0.21.0-1.5.5)

HttpDeflateStream::finish ― 圧縮ストリームを終了する

説明

public string HttpDeflateStream::finish ([ string $data ] )

圧縮ストリームを終了します。圧縮ストリームは、 終了した後でも再開することが可能です。

パラメータ

data

圧縮するデータ。

返り値

圧縮されたデータの最後の部分を返します。
