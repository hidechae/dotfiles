acosh

(PHP 4 >= 4.0.7, PHP 5)

acosh ― 逆双曲線余弦（アークハイパボリックコサイン）

説明

float acosh ( float $arg )

arg のアークハイパボリックコサインを返します。 つまり、ハイパボリックコサインが arg となる値です。

注意: この関数は Windows 環境にはまだ実装されていません。

パラメータ

arg

処理する値。

返り値

arg のアークハイパボリックコサインを返します。
