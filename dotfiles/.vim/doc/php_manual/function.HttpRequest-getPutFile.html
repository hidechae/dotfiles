HttpRequest::getPutFile

(PECL pecl_http:0.10.0-1.5.5)

HttpRequest::getPutFile ― PUT ファイルを取得する

説明

public string HttpRequest::getPutFile ( void )

事前に設定された PUT ファイルを取得します。

返り値

現在設定されている PUT ファイルへのパスを含む文字列を返します。
