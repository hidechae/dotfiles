mcrypt_enc_get_key_size

(PHP 4 >= 4.0.2, PHP 5)

mcrypt_enc_get_key_size ― オープンされたモードでサポートされる最大キー長を返す

説明

int mcrypt_enc_get_key_size ( resource $td )

この関数は、暗号化記述子 td で指定した アルゴリズムでサポートされる最大キー長をバイト単位で返します。
