rar_open ― Rar アーカイブをオープンする

説明

resource rar_open ( string $filename [, string $password ] )

指定した Rar アーカイブをオープンし、Rar ファイルリソースを返します。

パラメータ

filename

Rar アーカイブへのパス。

password

必要に応じて、プレーンテキストのパスワード。

返り値

rar_open() は成功した場合に Rar ファイルリソース、 エラー時には FALSE を返します。
