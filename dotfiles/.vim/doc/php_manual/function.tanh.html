tanh

(PHP 4 >= 4.0.7, PHP 5)

tanh ― 双曲線正接（ハイパボリックタンジェント）

説明

float tanh ( float $arg )

arg のハイパボリックタンジェントを返します。 これは sinh(arg)/cosh(arg) で定義されます。

パラメータ

arg

処理する引数。

返り値

arg のハイパボリックタンジェントを返します。
