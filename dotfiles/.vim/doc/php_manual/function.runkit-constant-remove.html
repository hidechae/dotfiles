runkit_constant_remove

(PECL runkit:0.7-0.9)

runkit_constant_remove ― 定義済みの定数を削除する

説明

bool runkit_constant_remove ( string $constname )

パラメータ

constname

削除する定数名。グローバル定数を表す文字列、あるいは classname::constname 形式でクラス定数を示す。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
