dba_close

(PHP 4, PHP 5)

dba_close ― DBA データベースを閉じる

説明

void dba_close ( resource $handle )

dba_close() は確立されたデータベースを閉じ、 により指定したデータベースハンドルの全てのリソースを開放します。

パラメータ

handle

dba_open() あるいは dba_popen() によって返されたデータベースハンドル。

返り値

値を返しません。

参考

  • dba_open()
  • dba_popen()
