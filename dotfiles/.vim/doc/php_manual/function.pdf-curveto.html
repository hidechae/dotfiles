PDF_curveto

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_curveto ― ベジエ曲線を描く

説明

bool PDF_curveto ( resource $p , float $x1 , float $y1 , float $x2 , float $y2 , float $x3 , float $y3 )

3 つの制御点を使用して現在の点からベジエ曲線を描画します。 成功した場合に TRUE を、失敗した場合に FALSE を返します。
