connection_aborted

(PHP 4, PHP 5)

connection_aborted ― クライアントとの接続が切断されているかどうかを調べる

説明

int connection_aborted ( void )

クライアントとの接続が切断されているかどうかを調べます。

返り値

クライアントとの接続が切断されている場合に 1、それ以外の場合に 0 を返します。
