mcal_delete_event

(PHP 4)

mcal_delete_event ― MCAL カレンダーからイベントを削除する

説明

bool mcal_delete_event ( int $mcal_stream , int $event_id )

mcal_delete_event() は event_id で指定した カレンダーイベントを削除します。

TRUE を返します。
