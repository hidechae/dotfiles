HaruFont::getUnicodeWidth

(PECL haru:0.0.1)

HaruFont::getUnicodeWidth ― フォントの文字の幅を取得する

説明

int HaruFont::getUnicodeWidth ( int $character )

フォントの文字の幅を返します。

パラメータ

character

文字のコード。

エラー / 例外

エラー時に HaruException をスローします。
