HaruDoc::useCNTEncodings

(PECL haru:0.0.1)

HaruDoc::useCNTEncodings ― 繁体字中国語エンコーディングを有効にする

説明

bool HaruDoc::useCNTEncodings ( void )

繁体字中国語エンコーディングを有効にします。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
