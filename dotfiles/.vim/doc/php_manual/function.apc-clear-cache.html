apc_clear_cache

(PECL apc:2.0-3.0.9)

apc_clear_cache ― APC キャッシュをクリアする

説明

bool apc_clear_cache ([ string $cache_type ] )

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

パラメータ

cache_type

cache_type が "user" の場合はユーザキャッシュがクリアされます。 それ以外の場合はシステムキャッシュ (キャッシュされたファイル) がクリアされます。
