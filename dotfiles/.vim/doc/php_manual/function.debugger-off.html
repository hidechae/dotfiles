debugger_off

(No version information available, might be only in CVS)

debugger_off ― PHP の内部デバッガを無効にする (PHP 3)

説明

int debugger_off ( void )

PHP の内部デバッガを無効にします。

返り値

成功した場合に 1、エラー時に 0 を返します。

注意

この関数は PHP 3 でのみ有効です。
