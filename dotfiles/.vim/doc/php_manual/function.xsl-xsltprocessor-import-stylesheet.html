XSLTProcessor::importStylesheet ― スタイルシートを取り込む

説明

XSLTProcessor
void importStylesheet ( DOMDocument $stylesheet )

このメソッドは XSLTProcessor に変換のためのスタイルシートを取り込みます。

パラメータ

stylesheet

取り込まれるスタイルシートを DOMDocument オブジェクトとして指定します。

返り値

値を返しません。
