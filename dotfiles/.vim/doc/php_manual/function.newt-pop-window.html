newt_pop_window

(PECL newt:0.1-1.1)

newt_pop_window ― トップウィンドウを画面から消去する

説明

void newt_pop_window ( void )

トップウィンドウを画面から消去し、ウィンドウが覆っていた部分を 再描画します。

返り値

値を返しません。
