deg2rad

(PHP 4, PHP 5)

deg2rad ― 度単位の数値をラジアン単位に変換する

説明

float deg2rad ( float $number )

この関数は、number の単位を度からラジアンに変換します。

パラメータ

number

度で表した角度。

返り値

number と同等な値をラジアンで表したものを返します。

例

Example#1 deg2rad() の例

<?php

echo deg2rad(45); // 0.785398163397
var_dump(deg2rad(45) === M_PI_4); // bool(true)

?>
