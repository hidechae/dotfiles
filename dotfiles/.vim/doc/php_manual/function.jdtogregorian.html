JDToGregorian

(PHP 4, PHP 5)

JDToGregorian ― ユリウス積算日をグレゴリウス日に変換する

説明

string jdtogregorian ( int $julianday )

ユリウス積算日を "月/日/年" の形式でグレゴリウス日を含む文字列に変換します。

パラメータ

julianday

ユリウス積算日を表す整数値。

返り値

グレゴリウス暦の日付を "月/日/年" 形式の文字列で返します。

参考

  • gregoriantojd()
  • cal_from_jd()
