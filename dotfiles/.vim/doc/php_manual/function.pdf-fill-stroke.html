PDF_fill_stroke

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_fill_stroke ― パスを塗りつぶし、パスの輪郭を描く

説明

bool PDF_fill_stroke ( resource $p )

現在のパスの内部を塗りつぶし色で塗りつぶし、輪郭を 輪郭色で描画します。 成功した場合に TRUE を、失敗した場合に FALSE を返します。
