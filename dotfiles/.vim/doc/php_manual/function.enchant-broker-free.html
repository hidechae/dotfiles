enchant_broker_free

(PECL enchant:0.1-1.0.1)

enchant_broker_free ― ブローカーリソースおよびその辞書を開放する

説明

bool enchant_broker_free ( resource $broker )

ブローカーリソースを、その辞書とともに開放します。

パラメータ

broker

ブローカーリソース。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
