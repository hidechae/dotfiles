symlink

(PHP 4, PHP 5)

symlink ― シンボリックリンクを作成する

説明

bool symlink ( string $target , string $link )

symlink()は、指定されたリンク名 link で既存のファイル target へのシンボリックリンクを作成します。

パラメータ

target

リンクの対象。

link

リンクの名前。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

注意

注意: この関数は Windows 環境にはまだ実装されていません。

参考

  • ハードリンクを作るには link()
  • readlink() や linkinfo()
