XMLReader::setRelaxNGSchemaSource

(PHP 5 >= 5.1.2)

XMLReader::setRelaxNGSchemaSource ― RelaxNG スキーマを含むデータを設定する

説明

XMLReader
bool setRelaxNGSchemaSource ( string $source )

検証の際に使用する RelaxNG スキーマを含むデータを設定します。

パラメータ

source

RelaxNG スキーマを含む文字列。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
