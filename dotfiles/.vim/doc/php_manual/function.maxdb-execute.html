maxdb_execute

(PECL maxdb:1.0)

maxdb_execute ― maxdb_stmt_execute() のエイリアス

説明

この関数は次の関数のエイリアスです。 maxdb_stmt_execute().

この関数エイリアスは非推奨であり、下位互換性維持のために残されています。 今後、PHP から削除される可能性がありますので、この関数を使用しないことを推奨します。
