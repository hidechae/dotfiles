asinh

(PHP 4 >= 4.0.7, PHP 5)

asinh ― 逆双曲線正弦（アークハイパボリックサイン）

説明

float asinh ( float $arg )

arg のアークハイパボリックサインを返します。 つまり、ハイパボリックサインが arg となる値です。

注意: この関数は Windows 環境にはまだ実装されていません。

パラメータ

arg

処理する角度。

返り値

arg のアークハイパボリックサインを返します。
