runkit_superglobals

(PECL runkit:0.7-0.9)

runkit_superglobals ― 登録されているスーパーグローバルを、数値添字の配列で返す

説明

array runkit_superglobals ( void )

返り値

登録されているスーパーグローバルを、数値添字の配列で返します。 スーパーグローバルとは、例えば _GET、_POST、_REQUEST、_COOKIE、_SESSION、_SERVER、_ENV、_FILES などです。
