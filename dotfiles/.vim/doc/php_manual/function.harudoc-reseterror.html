HaruDoc::resetError

(PECL haru:0.0.1)

HaruDoc::resetError ― ドキュメントハンドルのエラーの状態をリセットする

説明

bool HaruDoc::resetError ( void )

いったんエラーコードが設定されると、入出力処理を含む 大半の操作が実行できなくなります。 エラーへの対処を済ませたあとで処理を続けたい場合は、 この関数を実行してエラー状態をリセットする必要があります。

返り値

常に成功し、TRUE を返します。
