hw_GetParentsObj

(PHP 4)

hw_GetParentsObj ― 親のオブジェクトレコード

説明

array hw_getparentsobj ( int $connection , int $objectID )

オブジェクトレコードの添字配列に加えてオブジェクトでコードに関する 統計情報を含む連想配列を返します。各オブジェクトレコードは、 ID objectID を有するオブジェクトの 親に属しています。
