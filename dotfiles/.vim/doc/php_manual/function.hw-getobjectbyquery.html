hw_GetObjectByQuery

(PHP 4)

hw_GetObjectByQuery ― オブジェクトを検索する

説明

array hw_getobjectbyquery ( int $connection , string $query , int $max_hits )

サーバ全体のオブジェクトを検索し、オブジェクト ID の配列を 返します。最大マッチ数は、max_hits で制限 されます。max_hits が -1 にセットされた場合、 最大マッチ数は無制限となります。

クエリはインデックス付き属性でのみ動作します。

hw_getobjectbyqueryobj() も参照ください。
