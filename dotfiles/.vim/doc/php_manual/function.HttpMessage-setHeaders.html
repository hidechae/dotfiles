HttpMessage::setHeaders

(PECL pecl_http:0.10.0-1.5.5)

HttpMessage::setHeaders ― ヘッダを設定する

説明

public void HttpMessage::setHeaders ( array $headers )

新しいヘッダを設定します。

パラメータ

headers

新しい HTTP ヘッダを含む連想配列。メッセージの既存の HTTP ヘッダは、すべて上書きされます。
