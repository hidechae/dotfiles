printf

(PHP 4, PHP 5)

printf ― フォーマット済みの文字列を出力する

説明

int printf ( string $format [, mixed $args [, mixed $... ]] )

format にしたがって、出力を生成します。

パラメータ

format

format についての説明は sprintf() を参照ください。

args

...

返り値

出力した文字列の長さを返します。
