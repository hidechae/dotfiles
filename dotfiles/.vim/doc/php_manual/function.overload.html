overload

(PHP 4 >= 4.2.0)

overload ― クラスのプロパティおよびメソッドに関してオーバーロードを可能にする

説明

void overload ( string $class_name )

overload() 関数は、 class_name で指定されたクラスのプロパティと メソッドコールをオーバーロードします。

パラメータ

class_name

オーバーロードするクラス名。

返り値

値を返しません。

例

このパートの導入部にある 例 を参照ください。
