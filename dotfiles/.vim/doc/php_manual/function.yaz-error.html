yaz_error

(PHP 4 >= 4.0.1, PECL yaz:0.9-1.0.9)

yaz_error ― エラーの内容を返す

説明

string yaz_error ( resource $id )

yaz_error() は、yaz_errno() で返された直近のエラー番号に対応する英語のメッセージを返します。

パラメータ

id

yaz_connect() が返す接続リソース。

返り値

id で表される(直近のリクエストの) サーバに関するエラーメッセージを返します。 直近の操作が成功した場合には空の文字列が返されます。
