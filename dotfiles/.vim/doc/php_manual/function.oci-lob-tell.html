OCI-Lob->tell

(PHP 5, PECL oci8:1.1-1.2.4)

OCI-Lob->tell ― ラージオブジェクトの内部ポインタの現在位置を返す

説明

OCI-Lob
int tell ( void )

LOB の内部ポインタの現在位置を取得します。

返り値

LOB の内部ポインタの現在位置を返します。 エラーの場合、FALSE を返します。
