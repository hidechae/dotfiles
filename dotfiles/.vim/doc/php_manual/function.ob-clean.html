ob_clean

(PHP 4 >= 4.2.0, PHP 5)

ob_clean ― 出力バッファをクリア(消去)する

説明

void ob_clean ( void )

この関数は、出力バッファの内容を消去します。

この関数は、ob_end_clean() のように出力バッファを破棄するわけではありません。

返り値

値を返しません。
