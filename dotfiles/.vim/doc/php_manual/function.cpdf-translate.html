cpdf_translate

(PHP 4, PHP 5 <= 5.0.5)

cpdf_translate ― 座標系の原点を設定する

説明

bool cpdf_translate ( int $pdf_document , float $x_coor , float $y_coor )

座標系の原点を、指定した場所に設定します。

パラメータ

pdf_document

cpdf_open() が返す、ドキュメントのハンドル。

x_coor

位置の x 座標。

y_coor

位置の y 座標。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
