fbsql_rows_fetched

(PHP 5 >= 5.1.0)

fbsql_rows_fetched ― 直近の文で影響をうけた行の数を取得する

説明

int fbsql_rows_fetched ( resource $result )

直近の文で影響をうけた行の数を取得します。

パラメータ

result

fbsql_query() あるいは fbsql_db_query() が返す結果 ID。

返り値

行の数を整数値で返します。
