HaruPage::getCMYKFill

(PECL haru:0.0.1)

HaruPage::getCMYKFill ― 現在の塗りつぶし色を取得する

説明

array HaruPage::getCMYKFill ( void )

現在の塗りつぶし色を、4 つの要素 ("c", "m", "y" および "k") からなる配列で返します。

エラー / 例外

エラー時に HaruException をスローします。
