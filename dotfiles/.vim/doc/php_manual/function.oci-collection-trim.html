OCI-Collection->trim

(PHP 5, PECL oci8:1.1-1.2.4)

OCI-Collection->trim ― コレクションの最後から要素を切り取る

説明

OCI-Collection
bool trim ( int $num )

コレクションの最後から num 個の要素を切り取ります。

パラメータ

num

切り取る要素の数。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
