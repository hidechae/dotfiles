HaruDoc::save

(PECL haru:0.0.1)

HaruDoc::save ― ドキュメントを指定したファイルに保存する

説明

bool HaruDoc::save ( string $file )

ドキュメントを、指定したファイルに保存します。

パラメータ

file

ドキュメントを保存するファイル名。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
