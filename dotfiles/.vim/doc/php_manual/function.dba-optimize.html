dba_optimize

(PHP 4, PHP 5)

dba_optimize ― データベースを最適化する

説明

bool dba_optimize ( resource $handle )

dba_optimize() は、指定されたデータベースを最適化します。

パラメータ

handle

dba_open() あるいは dba_popen() によって返されたデータベースハンドル。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
