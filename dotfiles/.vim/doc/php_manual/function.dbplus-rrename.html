dbplus_rrename

(PHP 4 >= 4.0.7, PECL dbplus:0.9)

dbplus_rrename ― リレーションの名前を変更する

説明

int dbplus_rrename ( resource $relation , string $name )

dbplus_rrename() は、 relation の名前を name に変更します。

パラメータ

relation

dbplus_open() でオープンしたリレーション。

name

返り値

注意

警告
