OCI-Lob->rewind

(PHP 5, PECL oci8:1.1-1.2.4)

OCI-Lob->rewind ― 内部ポインタをラージオブジェクトの先頭に移動する

説明

OCI-Lob
bool rewind ( void )

内部ポインタをラージオブジェクトの先頭にセットします。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
