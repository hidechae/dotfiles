HttpQueryString::set

(PECL pecl_http:0.22.0-1.5.5)

HttpQueryString::set ― クエリ文字列パラメータを設定する

説明

public string HttpQueryString::set ( mixed $params )

クエリ文字列エントリを設定します。NULL を指定すると、変数を削除します。

パラメータ

params

追加するクエリ文字列パラメータ。

返り値

現在のクエリ文字列を返します。
