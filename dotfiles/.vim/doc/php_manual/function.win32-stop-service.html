win32_stop_service ― サービスを停止する

説明

int win32_stop_service ( string $servicename [, string $machine ] )

サービスを停止します。管理者権限を必要とします。

パラメータ

servicename

サービスの短い名前。

machine

オプションのマシン名。指定しなかった場合、ローカルマシンが使用されます。

返り値

成功した場合に WIN32_NO_ERROR 、失敗した場合に それ以外の win32 エラーコードを返します。
