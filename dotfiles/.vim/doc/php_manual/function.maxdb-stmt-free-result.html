maxdb_stmt_free_result

stmt->free_result

(No version information available, might be only in CVS)

stmt->free_result ― 指定したステートメントハンドルの結果を保存しているメモリを開放する

説明

手続き型

void maxdb_stmt_free_result ( resource $stmt )

オブジェクト指向型 (メソッド)

stmt
void free_result ( void )

maxdb_stmt_free_result() 関数は、 stmt パラメータで指定した結果メモリ (これは maxdb_stmt_store_result() によって確保されたものです) を開放します。

返り値

この関数は、なにも値を返しません。
