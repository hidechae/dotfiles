http_inflate

(PECL pecl_http:0.15.0-1.5.5)

http_inflate ― データを展開する

説明

string http_inflate ( string $data )

gzip、deflate 別名 zlib あるいは raw deflate エンコーディングで圧縮されたデータを展開します。

パラメータ

data

圧縮されたデータを含む文字列。

返り値

成功した場合はデコードした文字列、失敗した場合は NULL を返します。
