pcntl_wtermsig

(PHP 4 >= 4.0.7, PHP 5)

pcntl_wtermsig ― 子プロセスの終了を生じたシグナルを返す

説明

int pcntl_wtermsig ( int $status )

子プロセスを終了させたシグナルの数を返します。この関数は、 pcntl_wifsignaled() が TRUE を返す場合のみ有用です。

パラメータ

status

パラメータ status は、 pcntl_waitpid() が正常にコールされた際に得られます。

返り値

シグナル番号を表す整数値を返します。

参考

  • pcntl_waitpid()
  • pcntl_signal()
  • pcntl_wifsignaled()
