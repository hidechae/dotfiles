ps_fill_stroke

(PECL ps:1.1.0-1.3.5)

ps_fill_stroke ― 現在のパスを塗りつぶし、線を引く

説明

bool ps_fill_stroke ( resource $psdoc )

ps_lineto() のような描画関数によって事前に作成されたパスを塗りつぶし、描画します。

パラメータ

psdoc

ps_new() が返す、postscript ファイルのリソース ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
