pfpro_version

(PHP 4 >= 4.0.2, PHP 5 <= 5.0.5)

pfpro_version ― Payflow Pro ソフトウエアのバージョンを返す

説明

string pfpro_version ( void )

pfpro_version() は、Payflow Pro ライブラリの バージョンを文字列で返します。この文書を書いている時点では、 このバージョンは L211 です。

返り値

バージョン番号を文字列で返します。
