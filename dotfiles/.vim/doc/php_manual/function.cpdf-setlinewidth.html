cpdf_setlinewidth

(PHP 4, PHP 5 <= 5.0.5)

cpdf_setlinewidth ― 線幅を設定する

説明

bool cpdf_setlinewidth ( int $pdf_document , float $width )

線幅を設定します。

パラメータ

pdf_document

cpdf_open() が返す、ドキュメントのハンドル。

width

線幅を表す float 値。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
