is_finite

(PHP 4 >= 4.2.0, PHP 5)

is_finite ― 値が有限の数値であるかどうかを判定する

説明

bool is_finite ( float $val )

val が このプラットフォーム上で有効な有限値であるかどうかを調べます。

パラメータ

val

調べる値。

返り値

val が このプラットフォーム上の PHP の float 型で有効な範囲内の数である場合に TRUE、 それ以外の場合に FALSE を返します。
