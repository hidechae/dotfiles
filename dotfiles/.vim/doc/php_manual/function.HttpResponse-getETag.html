HttpResponse::getETag

(PECL pecl_http:0.10.0-1.5.5)

HttpResponse::getETag ― ETag を取得する

説明

static string HttpResponse::getETag ( void )

予測される、あるいは事前に設定された独自の ETag を取得します。

返り値

予測される、あるいは事前に設定された独自の ETag を、クォートされた文字列で返します。
