ovrimos_num_rows

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_num_rows ― update 命令により変更された行の数を返す

説明

int ovrimos_num_rows ( int $result_id )

update 操作で変更された行の数を取得します。

パラメータ

result_id

ovrimos_execute() あるいは ovrimos_exec() が返す結果 ID。

返り値

行の数を表す整数値、あるいはエラー時 FALSE を返します。
