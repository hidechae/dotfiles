OCI-Collection->append

(PHP 5, PECL oci8:1.1-1.2.4)

OCI-Collection->append ― コレクションに要素を追加する

説明

OCI-Collection
bool append ( mixed $value )

コレクションの最後に要素を追加します。

パラメータ

value

コレクションに追加する値を指定します。文字列か数値です。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
