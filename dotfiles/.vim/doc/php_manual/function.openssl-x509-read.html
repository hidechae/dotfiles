openssl_x509_read

(PHP 4 >= 4.0.6, PHP 5)

openssl_x509_read ― X.509 証明書をパースし、リソース ID を返す

説明

resource openssl_x509_read ( mixed $x509certdata )

openssl_x509_read() は、 x509certdata で指定した証明書をパースし、 その証明書用のリソースIDを返します。

パラメータ

x509certdata

返り値

成功した場合にリソース ID、失敗した場合に FALSE を返します。
