strval

(PHP 4, PHP 5)

strval ― 変数の文字列としての値を得ます

説明

string strval ( mixed $var )

var の string としての値を 返します。文字列への変換の詳細については、string の ドキュメントを参照ください。

var は、全てのスカラー値にできます。 strval() に配列あるいはオブジェクトは使用できません。
