openssl_free_key

(PHP 4 >= 4.0.4, PHP 5)

openssl_free_key ― キーリソースを開放する

説明

void openssl_free_key ( resource $key_identifier )

openssl_free_key() は、指定した key_identifier が指すキーをメモリから開放します。

パラメータ

key_identifier

返り値

値を返しません。
