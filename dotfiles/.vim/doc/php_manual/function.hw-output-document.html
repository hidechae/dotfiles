hw_Output_Document

(PHP 4)

hw_Output_Document ― hw_document を出力する

説明

bool hw_output_document ( int $hw_document )

BODY タグを除きドキュメントを出力します。

下位互換性のため、hw_OutputDocument() も使用可能です。しかし、この関数の使用は推奨されません。
