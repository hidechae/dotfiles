ibase_num_params

(PHP 5)

ibase_num_params ― プリペアドクエリのパラメータ数を返す

説明

int ibase_num_params ( resource $query )

この関数は、query で指定した プリペアドクエリのパラメータの数を返します。 ibase_execute() をコールする際に、 この数と同じだけのバインド引数が割り当てられている必要があります。

パラメータ

query

プリペアドクエリのハンドル。

返り値

パラメータの数を整数値で返します。

参考

  • ibase_prepare()
  • ibase_param_info()
