HaruPage::moveToNextLine

(PECL haru:0.0.1)

HaruPage::moveToNextLine ― テキストの位置を次の行の行頭に移動する

説明

bool HaruPage::moveToNextLine ( void )

テキストの位置を次の行の行頭に移動します。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
