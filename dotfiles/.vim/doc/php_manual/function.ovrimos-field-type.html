ovrimos_field_type

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_field_type ― 出力カラムの型を返す

説明

int ovrimos_field_type ( int $result_id , int $field_number )

出力カラムの型を返します。

パラメータ

result_id

ovrimos_execute() あるいは ovrimos_exec() が返す結果 ID。

field_number

1 からはじまるインデックス。

返り値

フィールドの型を表す整数値、あるいはエラー時に FALSE を返します。
