ps_stroke

(PECL ps:1.1.0-1.3.5)

ps_stroke ― 現在のパスを描画する

説明

bool ps_stroke ( resource $psdoc )

ps_lineto() のような関数で事前に組み立てられたパスを描画します。

パラメータ

psdoc

ps_new() が返す、postscript ファイルのリソース ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

参考

  • ps_closepath_stroke()
  • ps_fill()
  • ps_fill_stroke()
