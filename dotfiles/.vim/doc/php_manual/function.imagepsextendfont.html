imagepsextendfont

(PHP 4, PHP 5)

imagepsextendfont ― フォントを展開または圧縮する

説明

bool imagepsextendfont ( int $font_index , float $extend )

フォント(font_index )を展開または圧縮します。 パラメータextend の値が1より小さい場合、 フォントの圧縮が行われます。

パラメータ

font_index

extend

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

注意

注意: この関数は、PHP が --with-t1lib を指定してコンパイルされている場合のみ使用可能です。
