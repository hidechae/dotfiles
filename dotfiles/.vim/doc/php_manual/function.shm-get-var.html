shm_get_var

(PHP 4, PHP 5)

shm_get_var ― 共有メモリから変数を返す

説明

mixed shm_get_var ( int $shm_identifier , int $variable_key )

shm_get_var() は、 shm_identifier で指定した共有メモリセグメントから 変数 variable_key を読みこみます。 読み込んだ変数は、まだ共有メモリに存在します。

パラメータ

shm_identifier

共有メモリセグメント。shm_attach() から取得します。

variable_key

変数のキー。

返り値

指定したキーの変数を返します。
