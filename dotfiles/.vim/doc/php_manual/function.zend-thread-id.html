zend_thread_id

(PHP 5)

zend_thread_id ― 現在のスレッドの一意な ID を返す

説明

int zend_thread_id ( void )

この関数は、現在のスレッドの一意な ID を返します。

返り値

スレッドの ID を表す整数値を返します。

注意

注意: この関数が使用できるのは、PHP を ZTS (Zend Thread Safety) サポートつきでビルドした場合のみです。
