openal_source_play

(PECL openal:0.1)

openal_source_play ― ソースの再生を開始する

説明

bool openal_source_play ( resource $source )

パラメータ

source

Open AL (ソース) リソース (事前に openal_source_create() で作成したもの)。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
