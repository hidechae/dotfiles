mcal_popen

(PHP 4)

mcal_popen ― 永続的な MCAL 接続をオープンする

接続

int mcal_popen ( string $calendar , string $username , string $password [, int $options ] )

成功時に MCAL ストリームを、エラー時に FALSE を返します。

mcal_popen() は、指定した calendar が保存しているMCAL接続をオープン します。オプションの options が指定された場合、 options をメールボックスにも渡します。 内部イベント構造体のストリームも接続時に初期化されます。
