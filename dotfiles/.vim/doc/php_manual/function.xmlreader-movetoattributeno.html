XMLReader::moveToAttributeNo

(PHP 5 >= 5.1.2)

XMLReader::moveToAttributeNo ― 指定したインデックスの属性にカーソルを移動する

説明

XMLReader
bool moveToAttributeNo ( int $index )

指定した位置の属性にカーソルを移動します。

パラメータ

index

属性の位置。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
