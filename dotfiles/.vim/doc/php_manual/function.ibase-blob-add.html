ibase_blob_add

(PHP 4, PHP 5)

ibase_blob_add ― 生成された blob にデータを追加する

説明

void ibase_blob_add ( resource $blob_handle , string $data )

ibase_blob_add() は、 ibase_blob_create() で作成した blob にデータを追加します。

パラメータ

blob_handle

ibase_blob_create() でオープンした blob ハンドル。

data

追加するデータ。

返り値

値を返しません。
