getmypid

(PHP 4, PHP 5)

getmypid ― PHP のプロセス ID を取得する

説明

int getmypid ( void )

現在の PHP プロセスの ID を取得します。

返り値

現在の PHP のプロセス ID を返し、エラーの場合は FALSE を返します。

注意

警告

プロセス ID は一意ではなく、エントロピ源として優れたものではありません。 セキュリティが問題となる状況では、PID に頼らないようにしましょう。
