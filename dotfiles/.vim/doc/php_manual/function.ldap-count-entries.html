ldap_count_entries

(PHP 4, PHP 5)

ldap_count_entries ― 検索結果のエントリ数を数える

説明

int ldap_count_entries ( resource $link_identifier , resource $result_identifier )

直前の検索結果として保存されたエントリの数を返します。

パラメータ

link_identifier

ldap_connect() が返す LDAP リンク ID。

result_identifier

内部の LDAP 結果。

返り値

結果のエントリ数を返します。エラーの場合は FALSE を返します。
