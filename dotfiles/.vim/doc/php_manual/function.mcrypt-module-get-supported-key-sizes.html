mcrypt_module_get_supported_key_sizes

(PHP 4 >= 4.0.2, PHP 5)

mcrypt_module_get_supported_key_sizes ― オープンされたアルゴリズムでサポートされるキーのサイズを配列として返す

説明

array mcrypt_module_get_supported_key_sizes ( string $algorithm [, string $lib_dir ] )

指定したアルゴリズムでサポートされるキーのサイズを配列で返します。 この関数が空の配列を返した場合、1 から mcrypt_module_get_algo_key_size() の返り値の間のすべてのサイズのキーがアルゴリズムでサポートされます。 オプションのパラメータ lib_dir により、 システムで mode モジュールがある場所を指定することが可能です。

オープン暗号化モジュールで使用される mcrypt_enc_get_supported_key_sizes() も参照ください。
