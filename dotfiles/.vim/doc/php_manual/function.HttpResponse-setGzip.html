HttpResponse::setGzip

(PECL pecl_http:0.10.0-1.5.5)

HttpResponse::setGzip ― gzip を設定する

説明

static bool HttpResponse::setGzip ( bool $gzip )

送信されるエンティティを、その場で gzip 圧縮する処理を有効にします。

パラメータ

gzip

GZip 圧縮を有効にするかどうか。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
