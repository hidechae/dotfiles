dbmexists

(PHP 4)

dbmexists ― DBM データベース中に、キーに対応する値があるかどうかを調べる

説明

bool dbmexists ( resource $dbm_identifier , string $key )

キー key が指す値が存在すれば TRUE を返します。

パラメータ

dbm_identifier

dbmopen() が返す DBM リンク識別子。

key

キーを表す文字列。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
