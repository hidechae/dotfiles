date_date_set

(PHP 5 >= 5.1.0)

date_date_set ― 日付を設定する

説明

void date_date_set ( DateTime $object , int $year , int $month , int $day )
void DateTime::setDate ( int $year , int $month , int $day )

パラメータ

object

DateTime オブジェクト。

year

その日付の年。

month

その日付の月。

day

その日付の日。

返り値

成功した場合に NULL、失敗した場合に FALSE を返します。

  • date_isodate_set()
  • date_time_set()
