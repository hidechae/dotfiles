iconv_mime_decode_headers

(PHP 5)

iconv_mime_decode_headers 〗 剩眶の MIME ヘッダフィ〖ルドを办刨にデコ〖ドする

棱汤

array iconv_mime_decode_headers ( string $encoded_headers [, int $mode [, string $charset ]] )

喇根した眷圭は息鳞芹误を手します。その面咳には encoded_headers で回年した MIME ヘッダフィ〖ルドがすべて崔まれています。 デコ〖ド面にエラ〖が券栏した眷圭は FALSE を手します。

息鳞芹误の改」のキ〖がフィ〖ルド叹を山し、滦炳する妥燎がフィ〖ルドの 猛を山します。票じ叹涟のフィ〖ルドが剩眶赂哼する眷圭は、 iconv_mime_decode_headers() が极瓢弄に息戎つきの 芹误をつくり、叫附界にその芹误に掐れられます。

パラメ〖タ

encoded_headers

エンコ〖ドされたヘッダを山す矢机误。

mode

mode は、iconv_mime_decode_headers() が稍赖な妨及の MIME ヘッダフィ〖ルドに柳而した眷圭の 慷る神いを年盗します。笆布のビットマスクの寥み圭わせで回年が材墙です。

iconv_mime_decode_headers() で回年できるビットマスク
猛 年眶叹 棱汤
1 ICONV_MIME_DECODE_STRICT 回年すると、ヘッダは » RFC2047 で年盗されている筛洁に窗链洁凋する妨及でデコ〖ドされます。 このオプションはデフォルトでは痰跟になっています。なぜなら、坤の面には おかしなメ〖ルソフトが驴く赂哼し、それらは惮呈に骄わずに粗般った MIME ヘッダを栏喇するからです。
2 ICONV_MIME_DECODE_CONTINUE_ON_ERROR 回年すると、iconv_mime_decode_headers() は矢恕弄なエラ〖を痰浑し、デコ〖ド侯度を费鲁します。

charset

オプションの charset パラメ〖タは、冯蔡の 矢机セットを回年します。回年されなかった眷圭は iconv.internal_encoding が脱いられます。
