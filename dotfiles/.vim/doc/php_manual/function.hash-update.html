hash_update

(PHP 5 >= 5.1.2, PECL hash:1.1-1.5)

hash_update ― アクティブなハッシュコンテキストにデータを投入する

説明

bool hash_update ( resource $context , string $data )

パラメータ

context

hash_init() が返すハッシュコンテキスト。

data

ハッシュダイジェストに含めるメッセージ。

返り値

TRUE を返します。
