XMLReader::getParserProperty

(PHP 5 >= 5.1.2)

XMLReader::getParserProperty ― 指定したプロパティが設定されているかどうかを示す

説明

XMLReader
bool getParserProperty ( int $property )

指定したプロパティが設定されているかどうかを示します。

パラメータ

property

パーサオプション定数 のうちのひとつ。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
