finfo_close

(PECL fileinfo:0.1-1.0.4)

finfo_close ― fileinfo リソースを閉じる

説明

bool finfo_close ( resource $finfo )
finfo
__destruct ( void )

この関数は、finfo_open() がオープンしたリソースを閉じます。

パラメータ

finfo

finfo_open() が返す fileinfo リソース。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
